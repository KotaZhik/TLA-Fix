# Fix mod for That Level Again 1 & 2 (PC Only)
Fixes bugs and adds quality-of-life improvements for both TLA games.

> ⚠️ **REQUIRES ORIGINAL GAME:** This mod does NOT include game files. You must own the game on Steam:
> - [That Level Again](https://store.steampowered.com/app/3337220/That_Level_Again/)
> - [That Level Again 2](https://store.steampowered.com/app/4008290/That_Level_Again_2/)

## TLA 1 Fixes

- **Restored Erase Data button** in Options menu
- **Added Camera Expand button** in pause menu (excluded on levels 39, 43, 73, 81)
- **Text fixes** 13 languages included (Fix/TextFixes/TLA1/)
- **Menu selection bug** fixed after progress reset
- **Level 62:** All combinations now work (1-2-3-4, 12-3-4, 12-34, 1-23-4, 1-2-34)
- **Level 48:** Door opens without sound enabled

## TLA 2 Fixes

- **Restored Erase Data button** in Options menu
- **Text fixes** 4 languages included (Fix/TextFixes/TLA2/)
- **Level 43:** Camera expand blocked (prevents visual bugs)
- **Level 79:** Extended trigger hitbox for better detection
- **Level 80:** Death barrier prevents infinite falling
- **Fog reset** when returning to main menu

## Installation (TLA 1)

1. **Download the mod:** Get the latest `Fix.zip` from the [Releases page](https://github.com/KotaZhik/TLA-Fix/releases).
2. Download **BepInEx 5.4+** from the official website: [Download](https://github.com/BepInEx/BepInEx/releases)
3. Unpack the archive and copy the **BepInEx** folder into the `That Level Again` game folder
4. Launch the game, then close it. This will create the necessary folders
5. A **plugins** folder will appear inside `That Level Again\BepInEx`
6. Unpack **`Fix.zip`** into `That Level Again\BepInEx\plugins`
7. Launch the game and the mod will activate automatically!

## Installation (TLA 2)

For That Level Again 2, the installation is identical. Just copy the BepInEx folder into the TLA2 game folder and follow the same steps.
